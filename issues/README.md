# Issues

Each subdirectory in `/issues` corresponds to a single weekly issue of [Vectral Weekly](https://weekly.vectraltech.com).

Naming convention: `NNN-short-slug/` — e.g., `001-claude-opus-4-6/`, `002-gemini-3-tools/`.

Each issue folder contains:

- `README.md` — what this build does, how to run it, link to the published newsletter issue
- The actual working code (Node, Python, whatever fits the release)
- `prompts/` directory if any LLM prompts are involved
- `.env.example` if API keys are needed
